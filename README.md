# FitConnect

## Overview

FitConnect is a web-based fitness meetup platform developed as a Minimum Viable Product (MVP) using Agile Scrum methodology. The platform allows users to register, browse fitness events, join or cancel participation, manage their profile, and receive in-app notifications.

---

## Features

- User Registration
- User Login & Logout
- Dashboard
- Create Events
- Edit Events
- Delete Events
- View Event Details
- RSVP (Join / Cancel)
- Upcoming Sessions
- In-App Notifications
- Responsive User Interface

## Installation

1. Clone the repository.

git clone https://github.com/fitconnect-app1478/FitConnect.git

2. Copy the project into the XAMPP `htdocs` folder.

3. Start Apache and MySQL using XAMPP.

4. Import the database:

database/fitconnect.sql

5. Open the project:

http://localhost/FitConnect/

## Agile Development

The project was developed using Scrum with six sprints.

- Sprint 1 – Project Setup & Authentication
- Sprint 2 – Event Management
- Sprint 3 – Dashboard
- Sprint 4 – RSVP System
- Sprint 5 – Dashboard Improvements
- Sprint 6 – Notifications & UI Enhancements
